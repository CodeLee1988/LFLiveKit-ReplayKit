### Installation
	# To integrate LFLiveKit into your Xcode project using CocoaPods, specify it in your Podfile:
	# LFLiveKit-Recoder (2.7.5)
	
	$ pod 'LFLiveKit-Recoder'

	# Then, run the following command:
	$ pod install
